# SmartHire

SmartHire is a modern freelance platform that connects clients and freelancers through AI-generated contracts, real-time chat, milestone payments, and support for both Web2 (email/INR) and Web3 (wallet/crypto), ensuring secure, transparent, and smart collaboration.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **AI Assistant**: Get help with contracts and milestones.
- **Blockchain Escrow**: Secure payments using Ethereum Sepolia.
- **Milestone Tracking**: Clear progress monitoring for both parties.
- **Dispute Resolution**: Dedicated center for handling issues.
- **Customizable Dashboard**: Tailor your workspace to your needs.
